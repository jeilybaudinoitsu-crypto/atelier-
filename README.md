
# 🧵 Atelier - Plataforma de Diseño de Moda

¡Bienvenido a **Atelier**! Esta es una aplicación web interactiva pensada para diseñadores, entusiastas de la moda y creadores de ropa. Atelier permite explorar, personalizar y dar vida a tus ideas de diseño desde una interfaz intuitiva y moderna.
crado por Jeily Baudino

---

## 🚀 Características Principales

* **🔍 Búsqueda de Patrones:** Explora y encuentra los moldes perfectos para tus prendas a través de un sistema de búsqueda y filtrado dinámico.
* **🎨 Personalización de Colores:** Cambia la paleta de colores de los diseños en tiempo real para previsualizar diferentes combinaciones antes de la confección.
* **🖼️ Estampados Interactivos:** Sube tus propias imágenes, texturas o patrones gráficos para aplicarlos directamente sobre las prendas.
* **🖨️ Listo para Imprimir:** Exporta e imprime tus patrones a escala de manera rápida y sencilla para llevarlos directamente al taller.
